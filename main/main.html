HTML&CSS

HTML（HyperText Markup Language）とCSS（Cascading Style Sheets）は、ウェブページとウェブサイトを作成するために一緒に使われる二つの重要な技術です。HTMLはウェブページの構造を定義し、CSSはその外観やスタイルを定義します。

JavaScript

JavaScriptは、ウェブページに動的な要素やインタラクティブ性を追加するためのプログラミング言語です。HTMLがページの骨組みを構築し、CSSがスタイルを加えるのに対し、JavaScriptはウェブページに動きや機能を提供します。

UNIX Command

Unixコマンドとは、Unix系オペレーティングシステム（OS）上で実行されるコマンドラインインターフェースのコマンドのことを指します。これらのコマンドは、ファイル操作、システム管理、プロセス制御など、多くの基本的なタスクを実行するために使用されます。

GIT

Gitは、ソフトウェア開発で使用されるバージョン管理システムです。このツールは、プロジェクトの歴史を追跡することを可能にし、複数の開発者が同じプロジェクト上で協力して作業できるようにします。Gitを使用すると、コードの変更を効率的に管理し、異なるバージョン間での移動や比較、変更の統合が可能になります。

SQL

SQL（Structured Query Language）は、データベース管理システムでデータを操作・管理するために使用される標準的なプログラミング言語です。SQLを使用することで、データを効率的に検索、追加、更新、削除できます。

PHP

PHPは、「Hypertext Preprocessor」の略で、主にウェブ開発で使用されるサーバーサイドのスクリプト言語です。PHPは、動的なウェブページやウェブアプリケーションの作成に適しており、HTMLと組み合わせて使用されます。PHPコードはサーバー上で実行され、クライアント（ブラウザ）には結果のHTMLが送信されます。
